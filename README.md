# 📱 Digital Product Marketplace App (ByteMe)

A mobile application built with **Flutter** for buying and selling digital products such as e-books, design assets, and other downloadable content. This app provides a seamless and user-friendly experience for both buyers and sellers.

## 🚀 Features

### Users

* 🔐 User authentication (Sign up / Login)
* 👤 User profile management
* 🛍️ Browse, search, and filter digital products

### Buyer

* 🛒 Manage cart
* 💳 Purchase digital products (Midtrans integration)
* 📄 Product detail page
* ⭐ Product ratings and reviews
* 📥 Receive purchased products via email (SendGrid)

### Seller

* 💲 Sell digital products
* 📜 Manage products for sale

---

## 🛠️ Tech Stack

* **Frontend:** Flutter (Dart)
* **Payment Gateway:** Midtrans
* **Email Service:** SendGrid

> Backend and database technologies are currently under consideration and will be decided in future development.
